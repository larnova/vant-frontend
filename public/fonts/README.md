# Satoshi Font Setup

To use the Satoshi font in this application, please add the following font files to this directory:

- `Satoshi-Light.woff2` (weight: 300)
- `Satoshi-Regular.woff2` (weight: 400)
- `Satoshi-Medium.woff2` (weight: 500)
- `Satoshi-Bold.woff2` (weight: 700)
- `Satoshi-Black.woff2` (weight: 900)

## Where to get Satoshi font

Satoshi is a commercial font available from:
- [Fontshare](https://www.fontshare.com/fonts/satoshi) (free version available)
- [Satoshi Font Website](https://www.fonts.com/font/satoshi)

## Alternative: Using CDN

If you prefer to use Satoshi via CDN, you can update `app/layout.tsx` to use `@next/font` with a different approach, or add the font via CSS `@import` in `app/globals.css`.
